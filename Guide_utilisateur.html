Qu’est-ce que Pocket Tracker ?

Pocket Tracker est une application web conçue pour vous aider à gérer vos finances personnelles de manière intuitive et sécurisée. Elle vous permet de suivre vos opérations bancaires, de gérer vos comptes, de planifier votre budget, et de visualiser vos dépenses à travers différents graphiques et résumés.

Commencer rapidement

  • Commencez par ajouter une banque dans l’onglet “Banque / Comptes”
  • Ajoutez des comptes à cette banque, si besoin
  • Vous pouvez désormais visualiser votre tableau de bord en sélectionnant une banque. Il ne contient pour le moment aucune information
  • Ajouter des opérations avec l’onglet “Opération”
  • Vous pouvez également effectuer des virements interne si besoin dans l’onglet “Virement interne”
  • Vous pouvez aussi définir votre budget du mois dans l’onglet “budget”
  • Avec les opérations ajoutées, vous devriez désormais voir des informations dans votre tableau de bord.

Continuez d’ajouter vos entrées et sorties d’argent quotidiennement pour avoir un tableau de bord complet.

1 - Banque / Comptes

La première étape en ouvrant pour la première fois l’application, est d’ajouter une banque. Pour cela, ouvrez l’onglet “Banque / Comptes” dans la barre de navigation en haut de l’écran:

En cliquant sur cet onglet, vous ouvrez le module suivant :

Ce module vous permet d’ajouter / supprimer les Banques / Comptes. Il est composé d’une partie “Vos banques” et une partie “Vos Comptes”. Dans ces deux parties, vous trouverez une liste déroulante pour sélectionner une banque ou un compte, un emplacement de saisi pour ajouter une banque ou un compte, un bouton ➕ pour valider l’ajout, et un bouton 🗑️ pour supprimer une banque ou un compte sélectionné.


Pour ajouter une banque, il vous suffit de vous placer dans l’emplacement de saisi “Ajoutez une banque”.

Inscrivez le nom de votre banque puis cliquez sur le bouton ➕ situé à droite. Vous devriez désormais voir votre banque ajoutée dans la liste de sélection du dessus.

Lorsque vous ajoutez une nouvelle banque, un compte courant (CC) est automatiquement ajouté.

Pour visualiser les comptes d’une banque, vous devez sélectionner votre banque dans la liste des banques, puis vous trouverez dans la liste des comptes, l’ensemble des comptes associés à la banque sélectionnée.

Vous pouvez ajouter autant de comptes que vous le souhaitez en suivant le même principe que pour la partie “Vos banques”. Vous ne pouvez cependant pas ajouter de compte si vous n’avez pas sélectionné de banque.

Pour supprimer une banque, il vous suffit de la sélectionner puis d’appuyez sur le bouton 🗑️. Attention, cette manipulation est définitive et aura pour effet de supprimer tous les comptes associés à la banque sélectionnée, ainsi que toutes les informations et données associées.

Pour supprimer un compte, il vous suffit de sélectionner une banque, puis de sélectionner le compte que vous souhaitez supprimer, puis d’appuyer sur le bouton 🗑️. Attention, cette manipulation est définitive et aura pour effet de supprimer toutes les informations et données associées. Il vous ait cependant impossible de supprimer le compte courant.

Lorsque vous supprimer un compte contenant un solde d’argent, celui-ci est versé sur le compte courant.

2 - Opération

Le deuxième onglet “Opération” dans la barre de navigation en haut de l’écran ouvre le module suivant :

Dans ce module, vous pouvez ajouter toutes les entrées ou sorties d’argent. Il se compose d’une partie “Opération” pour ajouter des opérations et d’une partie “Historique des opérations du jour” pour visualiser les opérations que vous avez réalisé aujourd’hui et les supprimer si besoin.

Pour ajouter une opération, vous devez tout d’abord sélectionner une banque dans la liste déroulante. Vous devriez y trouver les banques que vous avez ajouté. Si vous n’en avez pas encore ajouté, veuillez ouvrir l’onglet “Banque / Comptes” et ajouter une banque. Pour plus d’informations, veuillez vous diriger vers la section 1 - Banque / Comptes.

Vous pouvez ensuite :

  • Ajouter une description à cette opération dans “Détail de l’opération” (optionnel)
  • Ajouter le montant de l’opération (obligatoire)
  • Sélectionner ➕ ou ➖ pour définir si cette opération est une entrée ou une sortie d’argent. Par défaut ➕ sera sélectionnée pour une entrée d’argent.
  • Sélectionner la catégorie de votre opération avec la liste de sélection (obligatoire)
  • Valider l’opération en appuyant sur le bouton ☑️

Toutes les opérations sont faites sur le compte courant. Lorsqu’une opération est faite, elle s’affiche dans la partie “Historique des opérations du jour”. Dans l’exemple ci-dessous, l’opération suivante a été ajoutée :

  • détail : Guide utilisateur
  • Montant : 50
  • Entrée / Sortie : ➕
  • Catégorie : Aide

Si je souhaite supprimer une opération réalisée, il me suffit d’appuyer sur le bouton 🗑️ .

Attention, seules les opérations effectuées le jour même sont affichées dans “Historique des opérations du jour”. Si vous effectuez une opération le lundi à 23h55, le mardi à 00h01, cette opération ne sera plus visible dans l’historique et vous ne pourrez plus la supprimer si besoin.

3 - Virement interne

Le troisième onglet “Virement interne” dans la barre de navigation en haut de l’écran ouvre le module suivant :

Ce module vous permet d’effectuer un virement interne, d’un compte vers un autre. Il se compose d’une partie “Virement interne” pour effectuer des virements internes et d’une partie “Historique des virements du jour” pour visualiser les virements internes que vous avez réalisé aujourd’hui et les supprimer si besoin.

Pour effectuer un virement interne, vous devez tout d’abord sélectionner une banque dans la liste déroulante. Vous devriez y trouver les banques que vous avez ajouté. Si vous n’en avez pas encore ajouté, veuillez ouvrir l’onglet “Banque / Comptes” et ajouter une banque. Pour plus d’informations, veuillez vous diriger vers la section 1 - Banque / Comptes.

Une fois la banque sélectionnée, vous devriez voir vos comptes ainsi que le montant de leur solde s’afficher dans les listes déroulantes “Compte débiteur” et “Compte créditeur”.

Vous devez ensuite :

  • Sélectionner un compte débiteur à gauche
  • Sélectionner un compte créditeur à droite (différent du premier)
  • Ajouter le montant du virement (montant ≤ solde du compte débiteur)
  • Valider le virement interne en appuyant sur le bouton ☑️

Lorsqu’un virement interne est fait, il s’affiche dans la partie “Historique des virements du jour”. Dans l’exemple ci-dessous, le virement suivant a été ajoutée :

  • Banque : EXEMPLE
  • Compte débiteur : CC
  • Compte créditeur : Livret A
  • Montant : 200

Si je souhaite supprimer un virement réalisé, il me suffit d’appuyer sur le bouton 🗑️.

Attention, seules les virements effectués le jour même sont affichées dans “Historique des virements du jour”. Si vous effectuez un virement le lundi à 23h55, le mardi à 00h01, ce virement ne sera plus visible dans l’historique et vous ne pourrez plus le supprimer si besoin.

4 - Budget

Le quatrième onglet “Budget” dans la barre de navigation en haut de l’écran ouvre le module suivant :

Cette onglet vous permet de planifier votre budget pour chaque catégorie. Il vous suffit de sélectionner une banque avec la liste de sélection. Vous devriez y trouver les banques que vous avez ajouté. Si vous n’en avez pas encore ajouté, veuillez ouvrir l’onglet “Banque / Comptes” et ajouter une banque. Pour plus d’informations, veuillez vous diriger vers la section 1 - Banque / Comptes.

Ensuite, inscrivez le montant du budget dans chaque catégorie puis validez à l’aide du bouton “Valider on budget ☑️”

⚠️ La somme des montants des catégories ne peut dépasser le montant de vos revenus.

⚠️ Toutes les catégories doivent être remplies pour valider votre budget. Si vous ne souhaitez pas définir de budget pour une catégorie, vous n’avez qu’à inscrire la valeur 0 dans cette catégorie.

Si vous souhaitez plus tard mettre à jour votre budget du mois, il vous suffit d’ouvrir à nouveau l’onglet de budget, de sélectionner votre banque, et de modifier les valeurs dans chaque catégorie.

5 - Dashboard

Maintenant que vous savez comment ajouter une banque, des comptes, des opérations, effectuer des virements internes et définir votre budget du mois, il ne vous reste plus qu’à visualiser et analyser ce qu’il se passe dans votre banque.

Vous pouvez désormais sélectionner la banque de votre choix pour afficher le tableau de bord.

Dans le tableau de bord, on retrouve 6️⃣ parties.

1️⃣

La première partie concerne les montants des soldes de la banque.

On y retrouve :

  • le montant de votre solde (montant de votre compte courant)
  • le montant total de vos avoirs (somme des montants de vos comptes)
  • le montant de vos autres comptes

2️⃣

La second partie concerne vos dépenses par rapport à votre budget.

On retrouve pour les 6 catégories de dépenses :

  • le montant de vos dépenses par rapport au budget défini
  • le nombre d’opérations effectuées pour la catégorie
  • la différence entre le budget défini et les dépenses

Lorsque la différence est positive, l’indicateur est vert, et dans le cas inverse l’indicateur est rouge.

Dans cette partie, vous avez également la possibilité de filtrer sur le mois et l’année.

S’afficheront dans ces filtres les mois et années pour lesquels vous avez effectué au moins une opération. Lorsque vous changez de mois ou d’année, les données se mettent à jour.

3️⃣

La troisième partie est le montant de votre épargne dans le mois. Il s’agit de la différence entre vos entrées et vos sorties d’argent.

Si l’indicateur est vert, votre épargne est positive, si il est rouge, votre épargne est négative.

4️⃣

La quatrième partie est l’évolution du total de vos avoirs.

Sur ce graphique, vous pouvez choisir de le visualiser en vision Mois ou Année en cliquant sur les deux boutons en haut à droite. Par défaut, la vision annuelle est activée.

Ce graphique évolue en fonction des filtres Mois et Année de la partie 2️⃣ “Où en êtes-vous dans votre budget ?”.

Exemple : Vous sélectionnez le mois de Janvier et l’année 2024

  • Si vous cliquez sur le bouton Mois, le graphique affichera l’évolution pour le mois de janvier 2024
  • Si vous cliquez sur le bouton Annee, le graphique affichera l’évolution pour l’année 2024, tout mois confondu

Ce graphique est interactif, vous pouvez donc survolez les points avec votre souris pour afficher le montant de vos avoirs à une date précise.

5️⃣

La cinquième partie vous montre la répartition de vos dépenses par catégories.

Sur ce graphique, vous pouvez également choisir de le visualiser en vision Mois ou Année en cliquant sur les deux boutons en haut à droite. Par défaut, la vision mensuelle est activée.

Ce graphique évolue en fonction des filtres Mois et Année de la partie 2️⃣ “Où en êtes-vous dans votre budget ?”.

Ce graphique est également interactif, n’hésitez pas à le survoler pour afficher le détail de chaque catégorie.

6️⃣

Enfin, la sixième partie vous permet de consulter l’historique de vos opérations et de vos virements internes.

Par défaut, l’application affiche l’historique des opérations. Sur chaque opération, vous retrouverez :

  • la date
  • la banque
  • le détail
  • la catégorie
  • le montant de l’opération

Vous avez la possibilité de n’afficher que certaines catégories en cliquant sur un ou plusieurs boutons de sélection :

Pour les désélectionner, cliquez à nouveau dessus.

L’historique affiche les 20 dernières opérations ou virements. Si vous souhaitez en afficher plus, cliquez sur le bouton ➕ en fin de page :

Pour afficher l’historique des virements internes, cliquez sur “VIREMENTS” en haut de cette partie.

Sur chaque virement vous retrouverez :

  • la date
  • la banque
  • le compte débiteur
  • le compte créditeur
  • le montant du virement interne

Chaque élément du tableau de bord se met à jour lorsque vous réalisez une action (ajout d’une opération, changement de budget, autre).

6 - Où sont stockés vos données ?

Toutes vos données sont stockés en local dans votre navigateur. Cela signifie que si vous ouvrez l’application sur un nouvel ordinateur, l’application ne contiendra pas vos données. Cette application ne contient pas de système de connexion, donc n’importe qui ayant accès à votre ordinateur, ouvrant cette application, accèdera aux données que vous aurez ajouté.

6.1 - IndexedDB

Vos données sont stockées dans la base de données IndexedDB. Vous pouvez accéder à cette base de données en faisant un clique sur la page de l’application puis “Inspecter”.

Rendez-vous ensuite dans l’onglet Application > IndexedDB > MaBaseDeDonnees

Vous trouverez ainsi toutes les données stockées de votre application. Si cela vous intéresse, vous êtes libre d’y jeter un œil.

6.2 - Import / Export

Vous avez la possibilité d’importer une base de données au format json en cliquant sur le bouton d’import en haut à droite de la barre de navigation.

En cliquant dessus, vous pouvez sélectionner un fichier json et confirmer votre choix.

⚠️ Cette action est définitive et entrainera le remplacement des données actuelles.

Vous avez également la possibilité d’exporter la base de données de l’application, au format json, en cliquant sur le bouton d’export en haut à droite de barre de navigation.

En cliquant dessus, cela lancera le téléchargement de la base de données.

Si vous souhaitez utiliser l’application sur un second ordinateur, je vous conseille d’effectuer cette action et d’importer sur votre second ordinateur votre base de données.
